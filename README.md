# Python Mini Projects 

A collection of structured Python command-line applications developed to strengthen core programming fundamentals and problem-solving skills.

## Projects Included

- Guess the Number  
- Random Password Generator  
- Expense Tracker  

## Technologies Used

- Python 3.13.7 
- Random and String modules  
- Conditional statements and loops  
- Data structures: lists and dictionaries


## To Run the project follow :
1.Clone the Repository : git clone https://github.com/Ayush17281/python-mini-projects.git

2.Navigate to the Project Folder : cd python-mini-projects

3.Run any project : python Expensetracker.py 
                    (or) 
                    python randompassword.py
                    (or) 
                    python guessthenumber.py
                    
## Purpose
To enhance logical thinking and implement real-world problem-solving using Python.
